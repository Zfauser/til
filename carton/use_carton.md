# Use Carton

**Carton** is a super tool managing Perl distribution dependencies:

```bash
$ cpan App::Carton
```

If you want to keep a clean install, consider using: [local::lib](https://metacpan.org/pod/local::lib)

An alternative to **Carton** is **Carmel**.
