# Use filemerge from the command line

Useful commandline tools for **[FileMerge]** a part of XCode.

```bash
% fmdiff <file1> <file2>
```

For use with Subversion:

```bash
% svn diff --diff-cmd fmdiff <fileunderversioncontrol>
```
