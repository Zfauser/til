# Install Homebrew

Install *Homebrew*, this missing package manager for *OSX*.

It gives you easy and maintainable access to cool tools like:

- `ack`
- `colortail`
- `diff-so-fancy`
- `fmdiff`
- `httpie`
- `irssi`
- `jq`
- `libxml2`
- `liquidprompt`
- `multitail`
- `node`
- `openssl`
- `pandoc`
- `rbenv`
- `sqlite`
- `tmux`
- `tmuxinator-completion`
- `tree`
- `wget`

Source: http://brew.sh/
